# Rate Card schema

The shape of the published `rate-card.json`. A document carries a top-level metadata block (release, attribution, currency) and a `models` array of per-model pricing and capability entries.

## Versions

| Version | Status | Released | Notes |
|---|---|---|---|
| [v1](./v1/) | draft | unreleased | [README](./v1/README.md) |

Each major version lives in its own directory and stays published indefinitely so previously released artefacts remain verifiable.
